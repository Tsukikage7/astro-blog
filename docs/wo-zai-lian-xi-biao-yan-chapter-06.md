# EP6：扮演者

今天上班，我遇到了一个不喜欢的客人。

他喝多了，一直骂骂咧咧。

按理说我应该不高兴。

但我笑着说：先生，您需要什么？

他说：再来一瓶酒。

我说：好的。

心里想的是：你喝死算了。

但我脸上还在笑。

女孩在旁边看着，等我走过来，她说：你笑得太标准了。

我说：什么意思？

她说：你笑的角度、弧度、时间，都是练过的。

我说：没有。

她说：你不用骗我，我看得出来。

我沉默了。

她说：你什么时候能不演？

我说：我不知道。

她说：你演得这么累，为什么不放下？

我说：我怕我不演了，别人会不喜欢我。

她说：那你觉得现在喜欢你的人，喜欢的是你，还是你的表演？

我愣住了。

她说：你想想，如果有一天你不演了，他们还会喜欢你吗？

我答不上来。

因为我不知道。

或者说，我不敢知道。

下班后，我坐在酒吧门口。

这次我没有拿烟。

我只是坐着，什么都不做。

女孩走过来，说：你现在看起来真实多了。

我说：为什么？

她说：因为你不笑了。

我下意识摸了摸脸，发现我真的没笑。

这是我二十五年来，第一次不笑。

她笑了，说：你看，你做到了。

我看着她，突然想说谢谢。

但我说不出口。

因为我还在演"懂礼貌"的人。

于是我什么都没说。

只是看着她。

她明白我想说什么，说：不用谢。

我点了点头。

第一次，我没有说"不客气"。

这种感觉，很奇怪。

但我觉得，我可能真的在改变了。
